# Go Vertical Slice

An example boilerplate for golang http service.

## Features
1. httprouter
1. sqlc
1. tern migrator

## Prerequisite

1. Go v1.20+
2. Docker

## Getting Started

1. Install dependencies

```sh
go mod download
```

2. Start server

```sh
go run main.go
```

## How to

### Run app

Just run the `run` target from `Makefile`

```sh
make run
```
